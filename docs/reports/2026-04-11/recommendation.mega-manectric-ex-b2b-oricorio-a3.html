Game Plan

This build is an electric aggression shell around Mega Manectric ex with Oricorio support, using maximum consistency and acceleration slots to pressure early. Core package is extremely rigid in samples: Electric Generator, Electrike, Mega Manectric ex, Oricorio, Poké Ball, and Professor's Research are all avg_count 2.0 with presence_rate 1.0, with Copycat as near-core support (avg_count 1.6667, presence_rate 1.0). Ranked strategy should emphasize fast setup and clean tempo conversion, because current performance is weak (win_rate_pct 43.43, rank 10, share_pct 1.73) and punishes inefficient sequencing.

Game Plan Stages

Open

Electric Generator

Prioritize opening hands with Electrike access plus one consistency card (Poké Ball or Professor's Research), and ideally Electric Generator to accelerate tempo. Sequence draw/search first to secure two-turn development instead of spending singleton interaction immediately. If Oricorio is available early, stage it as backup pressure so your board does not collapse when the first attacker is checked.

Mid

Electrike

Shift into continuous attack turns with Mega Manectric ex while using Oricorio to maintain threat density and reduce single-line dependency. Use Copycat/Research to keep resource flow stable and avoid empty turns. Time Cyrus and flex support cards only when they create concrete swing value. Given low current win rate, prioritize deterministic lines (guaranteed setup and attack continuity) over speculative, high-variance plays.

Close

Mega Manectric ex

In closing turns, commit resources to the line with the shortest guaranteed finish path, preserving one interaction card for the opponent's stabilization turn. If Mega Manectric ex is removed, pivot immediately to prepared secondary pressure rather than rebuilding from zero. Favor reliability over greed: guaranteed attacks and board presence are more valuable than delayed value plays in this underperforming shell.
