# Processed Artifact Schemas

This folder defines lightweight schema contracts for processed artifacts.

Schema version policy:
- Current version: `1.0.0`
- Backward-incompatible change: bump major
- Backward-compatible additive change: bump minor
- Metadata-only or typo fix: bump patch

Artifacts covered:
- `cards.normalized.json`
- `decks.normalized.json`
- `deck_cards.normalized.json`
- `validation/report.json`
- `meta_metrics/top_decks.json`
- `meta_metrics/top_cards.json`
- `meta_metrics/top_cards_by_archetype.json`
- `meta_metrics/trends_1d_7d.json`
- `meta_metrics/overview.json`
